# Safer 랜딩페이지

세이퍼(Safer) 공식 랜딩페이지입니다.

## 배포 방법 (GitHub Pages)

1. 이 저장소를 GitHub에 push
2. 저장소 → Settings → Pages
3. Source: **Deploy from a branch** 선택
4. Branch: `main` / `(root)` 선택 후 Save
5. 잠시 후 `https://[계정명].github.io/[저장소명]/` 에서 확인

## 파일 구조

```
/
├── index.html      ← 랜딩페이지 (이미지 base64 내장, 단일 파일)
├── .nojekyll       ← GitHub Pages Jekyll 처리 방지
└── README.md
```
